# 💬 MERN ICQ Chat App

פרויקט צ'אט בזמן אמת המבוסס על MERN Stack** (MongoDB, Express, React, Node.js).  
האפליקציה התכתבות – עם מערכת משתמשים, העלאת תמונת פרופיל, התחברות, צ'אט בזמן אמת, ו־Theme משתנה.

---

## 🧠 מה נלמד בפרויקט

| נושא | מה נלמד |
|------|-----------|
| 🗄️ **MongoDB** | יצירת מסד נתונים NoSQL, מודלים של משתמשים (`User`) והודעות (`Message`), שימוש ב־Mongoose. |
| 🧩 **Express.js** | בניית REST API, הגנה על ראוטים עם Middleware, ניהול אימות, חיבור Socket.io. |
| ⚛️ **React** | יצירת SPA מלאה, ניהול סטייט עם Zustand, שימוש ב־Hooks, עבודה עם רכיבים מודולריים. |
| 💬 **Socket.io** | יצירת צ'אט בזמן אמת: שליחת הודעות, עדכון Live של משתמשים מחוברים. |
| ☁️ **Cloudinary** | העלאת תמונות פרופיל ושמירה מאובטחת בענן. |
| 🧠 **Fullstack Deployment** | חיבור ה־Frontend וה־Backend, פריסה מלאה בשרת אחד. |

---

```bash
